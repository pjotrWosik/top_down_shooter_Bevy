# Project - Top-Down Game in Bevy + Avian

A simple top-down game built with Bevy 0.18 and Avian Physics 0.6.

## Tech Stack

- **Bevy** 0.18.1 — game engine
- **Avian3D** 0.6 — physics engine

## Features

- Top-down camera that follows the player
- Player rotates towards mouse cursor
- Physics-based movement with gravity
- Custom 3D player model (GLB)

## Controls

| Key | Action |
|-----|--------|
| W | Move forward |
| S | Move backward |
| A | Move left |
| D | Move right |
| Mouse | Rotate player |
| LPM | shoot the gun |
| R | reload the gun |
